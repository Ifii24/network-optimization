# 🗺️ Diagrams — Network Visualisations

### `MEDITLINE_network.drawio`
Editable network diagram of the MEDITLINE liner shipping instance. Open with [draw.io](https://app.diagrams.net) (free, browser-based) or the desktop app.

The diagram shows the 5-port Mediterranean network with edge costs and capacities:

```
         8/14
     A ──────── B
  1/2│        2/3│
7/16 │            │ 7/13
  1/2│        2/10│
     E ──────── D ──────── C
          1/8          8/9
```

**Ports:** A=Alexandria, B=Barcelona, C=Casablanca, D=Dubrovnik, E=Ephesus  
**Edge format:** `cost / capacity (TEU)`
